browser-check.js

Browser-check.js runs your code against the caniuse database and tells you what features it detected and if those features passed or failed the test. Browser-check.js is an open-source initiative and is currently in a pre-alpha state.

  1. Select the browser versions you would like to support
  2. Paste your code in the editor
  3. Click 'Check Code'
  4. Click on one of the results to view details
